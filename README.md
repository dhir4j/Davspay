# Davspay Solution - Modern UPI Payment Gateway

A stunning, cyberpunk-styled payment gateway website built with Next.js, TypeScript, and Styled Components. Features a modern design with light violet as the primary color, smooth animations, custom cursor effects, and full dark/light theme support. Complete with realistic developer documentation, API reference, SDK examples, and integration guides.
